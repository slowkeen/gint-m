# Гинт-М — Дизайн-система и визуальная стратегия
## Фокус: имидж, портфолио, новая цветовая палитра с нуля

---

## 1. ДИЗАЙН-СИСТЕМА С НУЛЯ

Поскольку брендбука нет и палитру строим с нуля — это одновременно и вызов, и возможность. Я предлагаю сразу создать систему, которая будет работать на современное позиционирование.

### 1.1 Концепция: «Архитектурная точность»

Гинт-М — генподрядчик с 20+ годами опыта, работающий с международными клиентами (Microsoft, SAP, Kaspersky). Дизайн должен транслировать:

- **Надёжность** — через стабильную, сдержанную палитру
- **Точность** — через чёткую типографику и геометричную сетку
- **Масштаб** — через крупные фотографии и обилие воздуха
- **Современность** — через анимации и актуальные паттерны

Ориентир — Pridex, но без копирования: Pridex позиционируется как «технологичные отделочные работы», Гинт-М — как «надёжный генподрядчик полного цикла». Тональность чуть теплее и основательнее.

Примечание: детали цветовой палитры и типографики перенесены в раздел Moodboard.

## 2. ПОРТФОЛИО — КЛЮЧЕВОЙ РАЗДЕЛ

Поскольку портфолио — приоритет, я детализировал предложения для этого раздела.

### 2.1 Каталог проектов (/projects/)

**Верх страницы:**
- Зона первого экрана: крупный заголовок «Проекты» + счётчик «500+ реализованных объектов»
- Фоновое фото лучшего проекта с затемнением

**Блок избранных проектов (новый):**
- 2–3 флагманских проекта крупно (на всю ширину или 50/50)
- Каждый: крупное фото + название + год + площадь + кнопка «Смотреть»
- Это задаёт «планку качества» для всего портфолио

**Фильтры:**
Визуально выделенные теги с количеством:
`Все (500+) | Офисные (120) | Спортивные (15) | Капстрой (30) | Ритейл (45) | Жилая (20)`

Добавить фильтр по площади (как у Pridex):
`до 500 м² | 500–1000 | 1000–5000 | 5000–10 000 | 10 000+`

**Сетка:**
- «Кирпичная» сетка (разные размеры карточек) — выглядит более «архитектурно»
- Или строгая сетка 3 колонки с эффектами при наведении
- Загрузка: «Показать ещё» вместо пагинации

**Блок клиентов внизу:**
Логотипы с переходом на фильтр по клиенту. Сейчас этот фильтр скрыт — предлагаю сделать его первоклассным элементом.

## 3. ИМИДЖ И ВИЗУАЛЬНОЕ ВПЕЧАТЛЕНИЕ

### 3.1 Первые 5 секунд: что видит посетитель

Сейчас: слайдер с проектами, непонятно кто и чем занимается.

Должно быть:
```
┌──────────────────────────────────────────────────┐
│                                                  │
│              [видео / таймлапс]                   │
│                                                  │
│     ГИНТ-М                                      │
│     Генеральный подрядчик                        │
│     полного цикла                                │
│                                                  │
│     20+ лет  ·  500+ проектов  ·  ISO 9001       │
│                                                  │
│     [ Обсудить проект ]   [ Портфолио → ]        │
│                                                  │
└──────────────────────────────────────────────────┘
```

За 5 секунд посетитель понимает: кто (генподрядчик), сколько опыта (20+ лет), масштаб (500+ проектов), что делать (обсудить или смотреть).

### 3.2 Тон визуальной коммуникации

| Аспект | Текущий сайт | Как должно быть |
|---|---|---|
| Общее ощущение | «Корпоративный сайт 2015» | «Современное архитектурное бюро» |
| Фотографии | Разнородные, разного качества | Единый стиль, профессиональная съёмка |
| Типографика | PT Sans, мелкий кегль | Outfit + Source Sans, крупные заголовки |
| Пространство | Плотная вёрстка | Обилие воздуха, 120px между секциями |
| Цвет | Невыразительный | Контрастная палитра с терракотовым акцентом |
| Анимации | Отсутствуют | Плавные, элегантные, сдержанные |
